College Advisement


Matriculate

Need help with the college application process? Check out Matriculate!
You'll get your personal college advisor (a current college student) that helps you plan out your essays, build your college list, and more (much, much more)!


Youtube

Here is a Youtube channel that covers many topics regarding college admissions, test prep, and scholarships.


Futurescape

If you have no clue what you want to study, that’s okay!
Take a personality test and explore different career paths through Futurescape by the American Student Assistance.


Common Data Set

The common data set (CDS) is an excellent tool to help you build your college list. A university’s common data set has accurate information about the school that are useful to you (e.g., academic profile of admitted students).
To learn more about the Common Data Set visit an article that explains the CDS in more details.
